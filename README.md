# Data Analytics in Cloud Computing (2025/2026)

This repository contains course materials, examples, and lab exercises for the Data Analytics in Cloud Computing course in the 2025/2026 academic year.

**Institution:** Algebra Bernays University  
**Programme:** Professional Master in Computer Engineering, sub-specialization in Data Science

## Repository structure

The repo is organized into subfolders, one per lab or exercise. Each folder includes code, datasets (if provided), and any supporting notes.

Example layout:
- `EX_1/` – Exercise 1 materials (notebooks, scripts, instructions)
- `EX_2/` – Exercise 2 materials
- `...`

> New exercises and labs will be added in separate folders.

## How to use

1. Open the folder for the exercise you’re working on (e.g. `EX_1/`).
2. Follow the instructions in that folder’s notebook / README / code comments.
3. Run the code in your own environment unless otherwise specified.

## Environment / requirements

Exercises will use one or more public cloud platforms and standard data analytics tools.

Some exercises may include:
- a `requirements.txt` (Python dependencies), and/or
- CLI scripts, and/or
- one or more `*.yaml` files (optional deployment templates)

## Notes
